# PF Lab 6 – Programming Fundamentals Lab

## Lab Objectives

1. **Iterate Over Data Structures**
   - Allow iteration over data structures like arrays, linked lists, or other collections without writing repetitive code for each element.

2. **Terminate on Specific Conditions**
   - Ensure loops execute only while a particular condition is met.
   - Halt execution once the desired outcome is achieved.

3. **Reduce Code Complexity**
   - Simplify complex operations by reducing the number of lines of code.
   - Provide a clean and logical structure for repeating tasks.

4. **Support Event-Driven Programming**
   - Enable continuous event-checking mechanisms in real-time applications.
   - Monitor inputs, actions, or triggers without prematurely terminating the program.

5. **Flexible Program Execution**
   - Allow loops to start, continue, or terminate based on condition evaluation.
   - Use control statements such as `break` and `continue` effectively.
